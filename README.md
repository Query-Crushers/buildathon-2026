# 🛡️ SwitchGuard Analytics
**Real-time Fraud Detection & Transaction Security Layer**

Built by **Team Query Crushers** for the **2026 Enyata x Interswitch Buildathon**.

---

## 🚀 Overview
SwitchGuard Analytics is a high-performance security dashboard designed to provide a data-driven layer of protection for digital transactions. By analyzing transaction metadata in real-time, the platform identifies suspicious patterns and helps prevent fraud before it impacts the bottom line.

This project was built to solve the growing complexity of fraud in the African fintech ecosystem, specifically optimized for integration with **Interswitch's** payment infrastructure.
